### Do we need to work so much?

There are bullshit jobs. But there are also jobs which a lot of the time there is bullshit. 

Do shops need to be manned by so many people? Do they need to be manned all the time?

There are also jobs which become bullshit once other concepts are changed e.g. money removed.
